The attributes package provides the interfaces that represent attribute entities of a TestDescription. The attributes package describes properties that may determine whether a test gets built or executed, and how the tests are built or executed.

The attributes interfaces express the typed attribute entities for TestGroup, TestCase, and Library. The attributes package also contains some interfaces that represent components of attributes.

The attributes package contains classes that will exist when a TestDescription XML file is parsed, or a programmer can synthesize a TestDescription using the AttributesFactory create methods.
